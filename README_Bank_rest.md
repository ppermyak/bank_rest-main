# Bank REST API

REST API для управления банковскими картами и переводами

## Технологии

* Java 17
* Spring Boot 3.2.2
* Spring Security + JWT (Аутентификация и авторизация)
* Spring Data JPA (Работа с БД)
* PostgreSQL (Основная база данных)
* Liquibase (Миграции базы данных)
* MapStruct (Маппинг DTO/Entity)
* OpenAPI (Swagger) (Документация API)
* Docker Compose (Контейнеризация)

## Предварительные требования

* Docker & Docker Compose

## Быстрый запуск (Docker Compose)

Этот способ автоматически соберёт проект и запустит приложение вместе с базой данных PostgreSQL.

1. **Собрать и запустить проект:**
   ```bash
   docker-compose up --build
   ```

2. **Доступ к приложению:**
    * API доступно по адресу: `http://localhost:8080`
    * Документация **Swagger UI**: `http://localhost:8080/swagger-ui.html`

## Основной функционал

* **Аутентификация:** Регистрация, авторизация и получение JWT токена.
* **Карты:** Создание, блокировка, просмотр баланса и управление картами.
* **Переводы:** Перевод денежных средств между картами.
* **История:** Просмотр истории транзакций и переводов.
